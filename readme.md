# File Analyzer

## Project Description

This java program allows to analyze a file by presenting : 
- name
- type
- extention
- mime type
- size
- authenticity

See : video folder

## How to run the project 

### Linux terminal 

run :

```
javac GUI.java
java GUI
```

### Eclipse

Import the project and run as GUI.


## How to use the program

Select a file that you want to test.
